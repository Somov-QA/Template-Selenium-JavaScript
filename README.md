# Проект автоматизированного тестирования Selenium (JavaScript & NodeJS)

<br>Скачать и установить NodeJS https://nodejs.org
<br>Скачать и установить Visual Studio Code https://code.visualstudio.com/
<br>Перейти в NPM и найти Selenium https://www.npmjs.com/package/selenium-webdriver
<br>Скачать Webdriver для Chrome http://chromedriver.storage.googleapis.com/index.html
<br>В папке testing выполнить команду npm init (потом  npm init -y)
<br>В папке testing создать файл index.js
<br>В папке testing выполнить команду npm install selenium-webdriver
<br>
<br>Чтобы запустить автотест нужно в папке testing выполнить команду node index
